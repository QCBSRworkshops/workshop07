Workshop 7: General and generalized linear mixed effects models

Chapter 2 Preparing for the workshop

To prepare for this workshop, you must do the following steps:

Download the R script and data required for this workshop:

  • R Script
  • Fish data
  • Arabidopsis data
  • Inverts data
  • GLMM Script (this is the script for [[r_workshop7|Workshop 6]] but lines 434-784 cover the GLMM section of this workshop)
  • glmm_funs (code to download for use in the GLMM section)

Make sure to load the following packages (see how in the R script):

  • ggplot2
  • lm4
  • AICcmodavg
  • MASS
  • vcdExtra
  • bbmle
  • DescTools
install.packages("ggplot2")
install.packages("lme4")
install.packages("AICcmodavg")
install.packages("MASS")
install.packages("vcdExtra")
install.packages("bbmle")
install.packages("DescTools")

library(ggplot2)
library(lme4)
library(AICcmodavg)
library(MASS)
library(vcdExtra)
library(bbmle)
library(DescTools)
